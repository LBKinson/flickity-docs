--- title: Style ---

Cell style

All sizing and styling of the cells are handled by your own CSS.

``` css /* cells will take up entire width of container */ .gallery-cell { width: 100%; height: 160px; margin-right: 10px;; } ```
``` css /* half-width cells with margin */ .gallery-cell { width: 50%; height: 160px; margin-right: 10px;; } ```

Cells can be different sizes. You can use any unit you like: percent, pixels, etc.

``` css .gallery-cell { width: 33%; height: 160px; margin-right: 10px; } .gallery-cell.size-180 { width: 180px; } .gallery-cell.size-large { width: 75%; } ```

You can use media queries to show different number of cells for different breakpoints.

``` css /* default full-width cells */ .gallery-cell { width: 100%; height: 160px; margin-right: 10px;; } @media screen and ( min-width: 768px; ) { /* half-width cells for larger devices */ .gallery-cell { width: 50%; } } ```

Previous & next buttons

Style and position previous & next buttons with CSS.

``` css /* big buttons, no circle */ .flickity-prev-next-button { width: 100px; height: 100px; background: transparent; } /* arrow color */ .flickity-prev-next-button .arrow { fill: white; } .flickity-prev-next-button.no-svg { color: white; } /* hide disabled button */ .flickity-prev-next-button:disabled { display: none; } ```
``` css /* smaller, dark, rounded square */ .flickity-prev-next-button { width: 30px; height: 30px; border-radius: 5px; background: #333; } /* arrow color */ .flickity-prev-next-button .arrow { fill: white; } .flickity-prev-next-button.no-svg { color: white; } /* position outside */ .flickity-prev-next-button.previous { left: -40px; } .flickity-prev-next-button.next { right: -40px; } ```

Page dots

Style and position page dots with CSS.

``` css /* position dots in gallery */ .flickity-page-dots { bottom: 0px; } /* white circles */ .flickity-page-dots .dot { width: 12px; height: 12px; opacity: 1; background: transparent; border: 2px solid white; } /* fill-in selected dot */ .flickity-page-dots .dot.is-selected { background: white; } ```
``` css /* position dots up a bit */ .flickity-page-dots { bottom: -22px; } /* dots are lines */ .flickity-page-dots .dot { height: 4px; width: 40px; margin: 0; border-radius: 0; } ```
