initialIndex

Zero-based index of the initial selected cell.

``` js initialIndex: 2 ```
{{! ----------------------------------------------------------------- }}

accessibility

Enable keyboard navigation. Users can tab to a Flickity gallery, and pressing left & right keys to change cells. Enabled by default accessibility: true.

``` js accessibility: false // disables being able to tab to Flickity // and keyboard navigation ```
{{! ----------------------------------------------------------------- }}

resize

Adjusts sizes and positions when window is resized. Enabled by default resize: true.

``` js resize: false // disables resize logic ``` ``` css /* gallery has fixed width */ .gallery { width: 300px; } ```

percentPosition

Sets positioning in percent values, rather than pixel values. If your cells do not have percent widths, we recommended percentPosition: false.

``` js // default percentPosition: false // sets cell position in percentages ``` ``` css /* cell width in percent */ .gallery-celll { width: 66%; } ```
``` js percentPosition: false // sets cell position in pixels ``` ``` css /* cell width in pixels */ .gallery-cell { width: 300px; } ```

sync

Syncs selecting cells between galleries. sync is useful if you want to use one Flickity gallery as navigation for another.

``` js $('.gallery-a').flickity({ // set sync as selector string sync: '.gallery-b' }) ``` ``` js // or $('.gallery-a').flickity({ // set sync as element sync: $('.gallery-b')[0] }) ```

sync requires the flickity-sync package. It already is included with flickity.pkgd.js, but not with Flickity as a stand-alone package. If you are using Browserify or RequireJS without flickity.pkgd.js, you need to install and require flickity-sync separately. See details in Extras.

UI

prevNextButtons

Creates and enables previous & next buttons. Enabled by default prevNextButtons: true.

``` js prevNextButtons: false // Disable previous & next buttons ```
{{! ----------------------------------------------------------------- }}

pageDots

Creates and enables page dots. Enabled by default pageDots: true.

``` js pageDots: false // Disable page dots ```
