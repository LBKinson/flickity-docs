Install

Download

  • JavaScript: flickity.pkgd.js uncompressed or flickity.pkgd.min.js minified
  • CSS: flickity.css

Install with Bower: bower install flickity --save

Install with npm: npm install flickity

There are several ways to initialize a Flickity instance.

You can use Flickity as a jQuery plugin: $('selector').flickity().

``` js // init Flickity $('#main-gallery').flickity({ // options cellAlign: 'left', contain: true }); ```

You can use Flickity with vanilla JS: new Flickity( elem ). The Flickity() constructor accepts two arguments: the container element and an options object.

``` js var elem = document.querySelector('#main-gallery'); var flicky = new Flickity( elem, { // options cellAlign: 'left', contain: true }); // element argument can be a selector string // for an individual element var flicky = new Flickity( '#main-gallery', { // options }); ```

You can initialize Flickity in HTML, without writing any JavaScript. Add js-flickity to the class of the container element. Options can be set with a data-flickity-options attribute.


Options set in HTML must be valid JSON. Keys need to be quoted, for example "cellAlign":. Note that the attribute value uses single quotes ', but the JSON entities use double-quotes ".

Cell style

All sizing and styling of the cells are handled by your own CSS.

``` css /* cells will take up entire container */ .gallery-cell { width: 100%; } ```
``` css /* half-width cells with margin */ .gallery-cell { width: 50%; margin-right: 10px; } ```

Cells can be different sizes. You can use any unit you like: percent, pixels, etc.

``` css .gallery-cell { width: 33%; margin-right: 10px; } .gallery-cell.size-180 { width: 180px; } .gallery-cell.size-large { width: 75%; } ```

You can use media queries to show different number of cells for different breakpoints.

``` css .gallery-cell { width: 100%; margin-right: 2%; } @media screen and ( min-width: 768px; ) { /* show 2 cells for larger devices */ .gallery-cell { width: 49%; } } ```

Options

cellAlign

Align cells within the container element.

  • Values: 'center', 'left', 'right'
  • Default: 'center'
``` js cellAlign: 'left' ```
``` js /* default 'center' */ ```
``` js cellAlign: 'right' ```

wrapAround

At the end of cells, wrap-around to the other end for infinite scrolling.

  • Values: true, false
  • Default: false
``` js wrapAround: true ```

contain

Contains cells to container element, so no excell scroll at beginning or end. Has no effect if wrapAround: true.

  • Values: true, false
  • Default: false
``` js contain: true ```

freeScroll

Enables content to be freely scrolled and flicked without aligning cells to an end position.

  • Values: true, false
  • Default: false
``` js freeScroll: true ```

Enable freeScroll and wrapAround and you can flick forever, man.

``` js freeScroll: true, wrapAround: true ```

autoPlay

Automatically advances to the next cell. If true, delay is 3 seconds. Set time between advances in milliseconds, for example autoPlay: 1500 will advance every 1.5 seconds.

Auto-playing will pause when mouse is hovered over, and resume when mouse is hovered off. Auto-playing will stop when the gallery is clicked or a cell is selected.

  • Values: true, false, Integer
  • Default: false
``` js autoPlay: true ```

imagesLoaded

Unloaded images have no size. If imagesLoaded is present on your site, Flickity can re-position cells once images have loaded.

  • Values: true, false
  • Default: false
``` js imagesLoaded: true ```

watching

You can enable and disable Flickity with CSS. watching option watches the content of :after of the gallery element. Flickity is enabled if :after content is 'flickity'.

IE8 and Android 2.3 do not support watching :after. Set watching: 'fallbackOn' to enable Flickity for these browsers.

  • Values: true, false, 'fallbackOn'
  • Default: false
``` js watching: true ``` ``` css /* enable Flickity by default */ .watching-demo:after { content: 'flickity'; display: none; /* hide :after */ } @media screen and ( min-width: 768px ) { /* disable Flickity for large devices */ .watching-demo:after { content: ''; } } ```

Other options

Defaults shown.

``` js accessibility: true, // enable keyboard navigation, pressing left & right keys cellSelector: undefined, // specify selector for cell elements draggable: true, // enables dragging & flicking initialIndex: 0, // zero-based index of the initial selected cell percentPosition: true, // sets positioning in percent values, rather than pixels // Enable if items have percent widths // Disable if items have pixel widths, like images pageDots: true, // enables page dots prevNextButtons: true, // creates and enables buttons to click to previous & next cells resizeBound: true, // listens to window resize events to adjust size & positions rightToLeft: false // enables right-to-left ordering for right-to-left languages ```
